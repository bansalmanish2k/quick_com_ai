# Quick Commerce AI

An AI-powered quick commerce application built with React, Vite, and advanced AI models. 

## Features
- **AI-Powered Smart List:** Upload a picture of a handwritten shopping list and the AI will automatically add the items to your cart.
- **Dynamic Search:** Instantly search through available products.
- **Modern UI:** Premium design with interactive, auto-scrolling category marquee and glassmorphism UI elements.

## Setup Instructions
1. Clone the repository
2. Run `npm install`
3. Start the development server using `npm run dev`

## Technologies Used
- React (JavaScript / JSX)
- Vite
- Advanced Vision AI API
- Lucide React (Icons)
- Fuzzysort (Fuzzy matching search)
